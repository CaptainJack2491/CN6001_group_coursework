# CVD Clinic Digital Transformation - CN6001/CN5023 Group Coursework

## Project Overview

West London Cardiovascular Centre (WLCC) digital transformation project. The system replaces paper-based patient records with a cloud-hosted web portal for managing patient registration, appointments, clinical forms, vital signs recording, and prescription generation.

1. Patient Registration
2. Appointment Booking
